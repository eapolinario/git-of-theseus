git-of-theseus — in‑browser

Runs the cohort & author analyses entirely in your browser via Pyodide and isomorphic-git. Plots are rendered with Plotly.

About / limitations
  • The repository is cloned over HTTPS into in‑browser memory. Most public hosts (including GitHub) require a CORS proxy — this page uses https://cors.isomorphic-git.org by default.
  • Blame in the browser is implemented over isomorphic-git and is an approximation of git blame -w -C --follow: it does not perform copy/move detection across files. Results may differ slightly from the CLI on repos with heavy refactoring.
  • Large repos can exhaust the browser tab’s memory budget. If the run stalls, increase the sampling interval, narrow the branch, or run the CLI instead.
  • For exact parity with the CLI, install the package locally: pip install 'git-of-theseus[plot]' and run git-of-theseus-analyze /path/to/repo.
